# 📖 ScreenOCR 사용 가이드

## 🎯 빠른 시작

### 1. 앱 실행
- `ScreenOCR.exe` 더블클릭
- 우측 상단에 작은 창이 나타납니다

### 2. 화면 캡처
- **스페이스바** 누르기
- 자동으로 전체 화면이 캡처됩니다

### 3. 저장
- 저장 다이얼로그가 나타나면 파일명 입력
- "저장" 버튼 클릭

### 4. 완료!
- 이미지 파일(.png)과 텍스트 파일(.txt)이 저장됩니다

---

## 🖼️ 메인 윈도우

### 위치 및 크기
- **위치**: 모니터 우측 상단
- **크기**: 150px × 120px (작고 간편)
- **특징**:
  - 항상 최상위 표시
  - 드래그하여 이동 가능
  - 투명도 90%

### 표시 내용
```
┌─────────────────┐
│  📸 OCR Capture │
│                 │
│ 스페이스바를 누르면│
│ 화면이 캡쳐됩니다. │
│                 │
│  [종료: ESC]     │
└─────────────────┘
```

---

## ⌨️ 단축키

| 키 | 기능 | 설명 |
|---|---|---|
| **Space** | 캡처 시작 | 전체 화면 캡처 + OCR 처리 |
| **ESC** | 종료 | 앱 종료 (확인 다이얼로그 표시) |

> **참고**: 전역 핫키이므로 다른 앱 사용 중에도 작동합니다

---

## 📸 화면 캡처 프로세스

### 자동 처리 흐름

1. **스페이스바 누름**
   - "📸 캡처 중..." 메시지 표시

2. **앱 창 자동 숨김**
   - 캡처 이미지에 앱이 포함되지 않도록 자동 숨김
   - 0.3초 대기

3. **화면 캡처**
   - 모든 모니터를 포함한 전체 화면 캡처
   - 원본 화질 유지

4. **앱 창 다시 표시**
   - "⏳ OCR 처리 중..." 메시지 표시

5. **OCR 텍스트 추출**
   - 이미지 자동 전처리 (대비 향상, 샤프닝)
   - Tesseract OCR로 텍스트 추출
   - 한/영/일/중 자동 인식

6. **저장 다이얼로그 표시**
   - 이미지 미리보기
   - 추출된 텍스트 미리보기 (첫 100자)

## 💾 저장 다이얼로그

### 구성 요소

```
┌─────────────────────────────┐
│  📁 파일 저장                 │
├─────────────────────────────┤
│ 파일명: [________________]   │
│                              │
│ 저장 위치: [폴더 선택 버튼]    │
│                              │
│ 미리보기:                     │
│ ┌─────────────────────┐     │
│ │ (이미지 썸네일)       │     │
│ └─────────────────────┘     │
│                              │
│ 추출된 텍스트:                │
│ ┌─────────────────────┐     │
│ │ [첫 100자 미리보기]   │     │
│ └─────────────────────┘     │
│                              │
│     [저장]      [취소]        │
└─────────────────────────────┘
```

### 파일명 규칙

**기본 파일명**: `screenshot_YYYYMMDD_HHMMSS`
- 예시: `screenshot_20250122_143025`

**사용자 지정 가능**:
- 원하는 이름으로 변경 가능
- 확장자는 자동으로 추가됨 (.png, .txt)

### 저장 위치

**기본 경로**:
```
C:\Users\{사용자명}\Documents\ScreenOCR\
```

**변경 방법**:
1. "폴더 선택" 버튼 클릭
2. 원하는 폴더 선택
3. "확인" 클릭

---

## 📁 저장 파일 형식

### 1. 이미지 파일 (.png)

- **형식**: PNG
- **품질**: 원본 화질 (무손실)
- **용량**: 화면 해상도에 따라 다름
- **예시**: `screenshot_20250122_143025.png`

### 2. 텍스트 파일 (.txt)

- **인코딩**: UTF-8
- **형식**:
```
==========================================
OCR 추출 결과
==========================================
캡처 일시: 2025-01-22 14:30:25
이미지 파일: screenshot_20250122_143025.png
==========================================

[추출된 텍스트 내용]

==========================================
End of OCR Result
==========================================
```

## 🎨 OCR 정확도 향상 팁

### 1. 선명한 화면 캡처
- 글자가 선명하고 대비가 높은 화면 선택
- 흐릿하거나 작은 글자는 인식률 저하

### 2. 적절한 해상도
- 너무 작은 글자는 확대 후 캡처
- 권장: 12pt 이상 글자 크기

### 3. 배경과 대비
- 검은 글자 + 흰 배경 (또는 반대)
- 화려한 배경이나 그라데이션은 피하기

### 4. 정리된 레이아웃
- 글자가 정렬된 문서
- 기울어진 텍스트는 인식률 저하

### 5. 지원 언어
- 한국어 (kor)
- 영어 (eng)
- 일본어 (jpn)
- 중국어 번체 (chi_tra)

> 혼합된 언어도 자동으로 인식됩니다

## 📊 멀티모니터 지원

### 자동 감지
- 연결된 모든 모니터 자동 감지
- 전체 가상 화면 캡처

### 작동 방식
- 모니터 1 + 모니터 2 = 하나의 큰 이미지로 저장
- 각 모니터의 배치 그대로 캡처

## ❓ 자주 묻는 질문 (FAQ)

### Q1: 특정 영역만 캡처할 수 있나요?
A: 현재 버전은 전체 화면 캡처만 지원합니다. 필요한 부분만 잘라내려면 이미지 편집 도구를 사용하세요.

### Q2: OCR 결과가 정확하지 않아요
A: 다음을 확인하세요:
- Tesseract 언어팩 설치 확인
- 글자 크기와 대비 확인
- 선명한 화면인지 확인

### Q3: 저장 위치를 기본으로 변경할 수 있나요?
A: 현재 버전은 매번 선택해야 합니다. 다음 버전에서 기본 경로 설정 기능을 추가할 예정입니다.

### Q4: 다른 단축키로 변경할 수 있나요?
A: 현재 버전은 Space와 ESC로 고정되어 있습니다.

### Q5: 동영상도 캡처할 수 있나요?
A: 현재는 정지 화면(스크린샷)만 지원합니다.

## 🚨 문제 발생 시

### OCR이 실행되지 않음
1. Tesseract 설치 확인
2. 언어팩 설치 확인
3. 로그 파일(`app.log`) 확인

### 저장이 안 됨
1. 저장 폴더 권한 확인
2. 디스크 공간 확인
3. 파일명에 특수문자 사용하지 않기

### 앱이 응답하지 않음
1. ESC 키로 종료 시도
2. 작업 관리자에서 강제 종료
3. 재실행

## 💡 고급 사용 팁

- 캡처 후 저장 다이얼로그에서 "저장"하지 않고 "취소"하면 재캡처 가능

- 정기적으로 저장 폴더를 정리하세요
- 불필요한 파일 삭제
